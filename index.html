Self Stabilizing Spoon

An assistive device designed for Parkinson's patients, built using Arduino, MPU6050 motion sensor, and two servo motors. The system compensates for hand tremors by actively stabilizing the spoon, allowing users to eat with greater comfort and independence.

Two Stage Communication System

Designed and implemented a digital communication system on an FPGA, integrating both source encoding/decoding and channel encoding/decoding into a single hardware pipeline. The system is optimized for handheld devices, achieving low latency, improved efficiency, and reduced power consumption compared to separate implementations.

Underground Cable Fault Detector

A fault detection system for underground power cables, implemented using an Arduino Nano, VSWR processing unit, and a 162 LCD display. The system identifies the location of faults by measuring signal variations and alerts the user through a buzzer and display output.
